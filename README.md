# dotfiles

## Usage
Executing the command `./install.sh` will set up the following:

1. Anaconda3
    - Used with `conda`
2. AWS CLI
    - Choice to install
3. Aliases
    - `ls` for `ls -lah`
    - `b` for `cd ..`
    - `da` for current date and time
4. SSH
    - authorized keys
    - host config
